文件说明：
getraychou
获取weather.raychou.com's rss结果并输出

用法：
usage: ./getraychou [CITYID] [DAYS]
CITYID:城市的ID，可以在http://weather.raychou.com/查到。如：广州：http://weather.raychou.com/?/detail/59287，那么59287就是CITYID。
DAYS：需要显示的天数。
本程序不输入任何参数则默认输出广州三天的预报。

注意：
第一次运行请修改SHPATH为你所定义的文件夹。
